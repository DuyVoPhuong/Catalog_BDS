Guideline - Website Catalog Bất Động Sản (Phase 1)

1. Mục tiêu

Xây dựng website catalog giới thiệu dự án bất động sản/thiết kế nội thất theo mô hình SPA, không cần backend, tối ưu chi phí và deploy miễn phí.

2. Tech Stack

Thành phầnĐề xuấtLý do
FrameworkReact 19 + Vite + TypeScriptNhanh, nhẹ, dễ mở rộng
CSSTailwindCSSResponsive, utility-first
RouterReact Router DOMSPA routing
AnimationFramer MotionFade, scroll, hover
SliderSwiperHero & gallery
IconLucide ReactNhẹ, đẹp
GalleryYet Another React LightboxXem ảnh fullscreen
SEOReact Helmet AsyncMeta title/description
ContactEmailJS hoặc FormSubmitKhông cần backend

5. Quản lý dữ liệu

Lưu dữ liệu dạng TypeScript/JSON trong src/data.

export const projects = [{
 id:1,
 slug:"akari-binh-tan",
 title:"Akari Bình Tân",
 category:"Apartment",
 thumbnail:"/images/projects/akari/thumb.webp",
 gallery:["..."],
 description:"..."
}]
